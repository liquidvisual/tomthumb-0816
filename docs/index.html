Bootstrap 4

This codebase is running Bootstrap 4 (Alpha v3). Despite being an alpha, it's thoroughly tested and solid for a website of this size. It will also be easily upgradable in future.

Access Bootstrap Docs


Utility Classes

You will likely see lots of weird classes around the place. Eg. .m-b-0 - these are utility classes from Bootstrap which allow us to set arbitrary spacing between elements without having to hard code those values separately. Visit Utilities Here

Themes

Navigation text can be changed (lighter or darker) based on what hero image is being displayed. This is controlled manually by placing a theme class on the <body>

.theme-home-hero Applies light gray to page nav anchors.
.theme-light-hero Applies dark gray to page nav anchors.
